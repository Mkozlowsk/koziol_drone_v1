# STM32F411 FPV Drone
In progress..

## Project Description
Hobby FPV drone basing on a STM32F411vet6 uC. Still in progress.

### Features
- Flight controller with a gyroscope, accelerometer, magnetometer, compass, barometer and gps.
- PC FPV reciever
- 4 BLDC rotors drived by ESCs

### Technologies and Tools
- **KiCad**  
- **CubeIDE**
- **STM32F411**
